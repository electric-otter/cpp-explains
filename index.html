Understanding C++: A Basic Guide

Introduction to C++

C++ is a **high-level programming language** that was developed in 1983 by Bjarne Stroustrup at Bell Labs [1]. It is used for developing various applications [1]. This guide provides an overview of its fundamental concepts.

Basic Structure of C++ Programs

A C++ program typically includes header files, uses the `std` namespace, and has a `main` function [2]. The `main()` function is the starting point of program execution [2].

// Header files #include <bits/stdc++.h> // std namespace contains // various standard library components using namespace std; // main function is the starting //point of program execution int main () { // This is the section where // we write code statements return 0; } [2]

Comments

**Comments** in C++ are used to provide explanations of the code, making it easier for others to understand its functionality [3]. They are not executed by the compiler and can also be used to temporarily disable specific statements without deleting them [3].

There are two types of comments [3]:

  • Single-lined: Indicated by two forward slashes `//` [3].
    // This is a comment [3]
  • Multi-lined: Begin with `/*` and end with `*/` [3].
    /* This is multi-lined comment. The below statement will print GeeksforGeeks.*/ [3]

Variables

In C++, a **variable** is a container used to store data values [3]. Variables must be declared before they can be used, and multiple variables can be declared at once [3].

  • Naming Rules: A variable name can contain alphabets, digits, and underscores, but it must start with an alphabet or an underscore [3].
  • Identifiers: All variables must be given **unique names**, known as identifiers, so each variable can be uniquely identified [4].
  • Constants: **Constants** are fixed values that remain unchanged during program execution [4].
  • Syntax:
    // Declaring a single variable data_type var_name; // Declaring multiple variables data_type var1_name, var2_name, var3_name; [4]

Data Types

**Data types** define the kind of data a variable can store [4].

  • Integer (`int`): Used to store integers, typically taking 4 bytes of memory [4].
    int var = 123; [5]
  • Character (`char`): Used to store characters, taking 1 byte of memory [5].
    char var = 'a'; [5]
  • Floating Point (`float`): Used for single-precision floating-point numbers, taking 4 bytes of memory [5].
    float num = 1.23; [5]
  • Double (`double`): Used for double-precision floating-point numbers, taking 8 bytes of memory [5].
    double num = 1.2345; [5]
  • Boolean (`bool`): Used to store logical values, either `true` or `false` [5, 6].
    bool b = false; [6]
  • String (`string`): A collection of characters surrounded by double quotes, used to store words or sentences [6]. It is part of the Standard Library and requires including the `` header file [6].
    string str = "GeeksforGeeks"; [6]

Input and Output

  • Input from user: Achieved using `cin` from the `iostream` library [6, 7].
    int var; cin >> var; [7]
  • Output on the console: Achieved using `cout` from the `iostream` library [7].
    cout << “Hello World”; [7]
  • New Lines: Can be inserted using `\n` character or `endl` [7].
    cout << "Hello World! \n"; cout << "Hello World!" << endl; [7]

Conditional Statements

**Conditional statements** allow controlling the program's flow based on specific conditions, enabling the execution of particular code sections [7].

  • `if` statement: Executes a block of code only if the given condition is true [8].
    if (condition) { // Code to be executed if the condition is true } [8]
  • Nested `if` statement: An `if` statement inside another `if` statement [8].
  • `if-else` statement: Executes code in the `if` block if the condition is true, otherwise executes code in the `else` block [8, 9].
    if (condition) { // Code to be executed if the condition is true } else { // Code to be executed if the condition is false } [9]
  • `else-if` statement: Allows checking for multiple conditions sequentially [9].
  • Shorthand `if-else` (Ternary Operator `?:`): A concise way to write `if-else` statements [10].
    (condition) ? expression1 : expression2; [10]
  • `switch` statement: Evaluates an expression and compares its value with various cases, executing the code for the matching case [10, 11].
    • The **`break`** keyword exits the switch statement when a case matches [11].
    • The **`default`** keyword (optional) executes if none of the cases match the expression's value [11].
    switch (expression) { case value1: // Code to be executed if // expression matches value1 break; case value2: // Code to be executed if // expression matches value2 break; // ... default: // Code to be executed if // expression does not match any case break; } [11]

Loops

**Loops** are used to repeatedly execute a block of code multiple times [12].

  • `for` Loop: Executes a block of code a fixed number of times [12].
    for (initialization expr; test expr; update expr) { // body of the for loop } [12]
  • `while` Loop: Repeatedly executes a block of code as long as the given condition is true [12].
    while (condition) { // statements update_condition; } [12]
  • `do-while` Loop: Executes the block of code at least once, then checks the condition at the end of the loop body [13].
    do { // Body of do-while loop } while (condition); [13]

Arrays

An **array** is a data structure that stores a fixed number of elements of the same data type in contiguous memory locations [13].

  • Syntax: `dataType array_name[size];` [14].
  • Multi-Dimensional Arrays: Known as arrays of arrays, they store similar types of data in tabular form [15]. Two-dimensional arrays are the most commonly used [15].
    data_type array_name[size1][size2]....[sizeN]; [15]

Vectors

**Vectors** are dynamic array-like data structures that store elements of the same data type contiguously and can automatically resize [16]. They are part of the C++ Standard Template Library (STL) and require including the `` header file [16].

Commonly used Vector Functions:

  • `push_back()`: Inserts elements at the end [16].
  • `pop_back()`: Removes elements from the end [16].
  • `clear()`: Removes all elements [16].
  • `empty()`: Checks if the vector is empty [16].
  • `at(i)`: Accesses the element at a specified index `i` [16].
  • `front()`: Accesses the first element [16].
  • `back()`: Accesses the last element [16].
  • `erase()`: Removes an element at a specified position [16].

References and Pointers

  • References: Provide an **alias for an existing variable** and are declared using the `&` operator [17].
    int var = 12; // A reference variable to var int &ref = var; [17]
  • Pointers: Variables that **store the memory address of another variable** [17]. They are created using the `*` operator and assigned an address using the address-of operator `&` [17].
    int i = 3; // A pointer to variable i or "stores the address of i" int *ptr = &i; [18]

Functions

**Functions** are reusable blocks of statements that perform a specific task, providing code reusability and modularity to a program [18, 19].

  • Syntax for Function Declaration: `return_type function_name(parameters);` [18].
  • Syntax for Function Definition: Contains the actual statements to be executed when the function is called [18, 20].
    return_type function_name(parameters) { // function body // code to be executed // return statement (if applicable) } [18]
  • Function Call: Invokes the function by providing arguments [21]. A function call must always be after either function definition or function prototype [20].
    function_name(arguments); [22]
  • Return Type: The data type of the optional value returned by the function; only one value can be returned [20, 21].
  • Parameters: The input data passed to the function by the caller [20, 21].

String Functions

C++'s Standard Template Library provides several functions for string manipulation [23].

  • `length()`: Returns the length of a string [23].
  • `substr(pos, len)`: Extracts a substring from a given string, starting at `pos` with `len` length [24].
  • `append()`: Appends a string to the end of another string [25].
  • `compare()`: Compares two strings lexicographically [25].
  • `empty()`: Checks if a string is empty [26].

Math Functions

The C++ Standard Library also includes common math functions [27].

  • `min(x, y)`: Returns the minimum value of x and y [27].
  • `max(x, y)`: Returns the maximum value of x and y [27].
  • `sqrt(x)`: Returns the square root of x [27].
  • `ceil(x)`: Rounds up the value x to its nearest integer [27].
  • `floor(x)`: Rounds the value of x downwards to the nearest integer [27].
  • `pow(x, n)`: Returns the value x raised to the power of n [27].

Object-Oriented Programming (OOP)

**Object-oriented programming** generally involves storing data in the form of **classes and objects** [28].

  • Class: A user-defined data type that contains its data members and member functions; it acts as a blueprint for objects with similar attributes and behavior [28].
  • Objects: An instance or a variable of a class [28].

Pillars of OOPs:

  • Encapsulation: Wrapping up data and methods together within a single entity [28]. In C++, classes are used for encapsulation [28].
  • Abstraction: Showing only necessary details while hiding internal implementation details [29].
  • Inheritance: Deriving properties from a **Parent class** to a **Child class** for code reusability [29, 30].

    Types of Inheritance [29, 31]:

    • Single Inheritance: A derived class inherits from a single base class [29, 31].
    • Multiple Inheritance: A derived class inherits from multiple base classes [29, 31].
    • Multilevel Inheritance: A derived class inherits from another derived class [29, 31].
    • Hierarchical Inheritance: More than one derived class inherits from a single base class [29, 31].
    • Hybrid (Virtual) Inheritance: Combines more than one type of inheritance [29, 31].
  • Polymorphism: Providing different functionalities to functions or operators with the same name [30].

    C++ offers two types of polymorphism [30]:

    • Compile-time Polymorphism: Achieved using Operator overloading and Function overloading [30].
    • Runtime Polymorphism: Achieved using Function overriding and Virtual Functions [30].

File Handling

**File handling** in C++ involves reading data from a file and manipulating it [32].

File Handling Operations:

  • Open a file: Using the `open()` member function of the `ofstream` class [32].
  • Read a file: Using the `getline()` member function of the `ifstream` class [32].
  • Write to a file: Using the `<<` operator with an `ofstream` object [32].
